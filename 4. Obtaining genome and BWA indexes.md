#get the genome from nbci
input file: fna.gz
then we unzip it to fna 
Before alignment, BWA creates a searchable index of the genome. (creating a dictionary/index for fast lookup
organizing the genome for rapid read mapping)

```
wget <link to .fna.gz file on ncbi>
gunzip <genome file name> 

#PBS -N genome_indexing
#PBS -l select=1:ncpus=1:mem=32gb:scratch_local=32gb
#PBS -l walltime=01:00:00

module load bwa-mem2

bwa-mem2 index /storage/brno2/home/kat/GCA_964341245.1_ilLimPopu1.hap1.1_genomic.fna
```
Output files

BWA creates several index files
e.g
GCA_964341245.1_ilLimPopu1.hap1.1_genomic.fna.0123
GCA_964341245.1_ilLimPopu1.hap1.1_genomic.fna.amb

GCA_964341245.1_ilLimPopu1.hap1.1_genomic.fna.bwt.2bit.64
GCA_964341245.1_ilLimPopu1.hap1.1_genomic.fna.pac

These are specialized binary/text index files 

the genome is ready to be used for alignment.
